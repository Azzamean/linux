Name: CM Glossary Search Widget
Contributors: CreativeMindsSolutions
Donate link: http://www.cminds.com/
Tags: search widget, widget, glossary search, glossary widget
Requires at least: 4.4
Tested up to: 5.7.2
Stable tag: 1.1.0

Lets you add search widget to your Tooltip Glossary plugin.


== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure plugin.
